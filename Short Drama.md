# Short Drama

## 用途

专为即梦 Agent 设计的短剧开头生成工具。通过引导用户选择类型和输入故事情节，生成抓人眼球的 12 秒短剧开头提示词，并最终生成视频。本 Skill 内置了生成问题、表格以及视频的能力。

## 使用方式

1. 在即梦 Agent 中上传或加载本 Markdown 文档。
2. 用户提出短剧开头、剧情钩子或视频脚本需求后，先调用即梦 Agent 的「生成问题」能力收集选择。
3. 用户确认类型和核心故事后，生成 12 秒分镜表格。
4. 用户确认分镜后，输出完整视频提示词。
5. 用户确认提示词后，调用即梦 Agent 的「生成视频」能力。

## 触发条件

- 用户想要创作短剧开头
- 用户提到"短剧"、"开头"、"钩子"、"爆款"等关键词
- 用户想要生成视频脚本

## 工作流程

### 第一步：类型选择与需求收集

**⚠️ 必须使用即梦 Agent 的「生成问题」功能，以可点击选项卡片的形式向用户提问，不要输出纯文字。**

**问题 1：选择你想要的短剧类型**（生成为可点击选项卡片）

选项：
- 重生
- 豪门
- 修仙
- 悬疑
- 末世
- 校园
- 恋爱
- 奇幻
- 无限流
- 古装
- 自定义

题材选项只显示上面的短词，不要附加解释、案例、箭头、长描述或表情符号。

**问题 2：选择或描述你的核心故事**（生成为可点击选项卡片）

根据用户在问题 1 中选择的类型，AI 自动生成 3-5 个该类型下的热门故事设定作为选项，供用户直接选择。同时提供"自定义"选项让用户自由输入。

核心故事设定里只保留一个自定义入口，选项名称统一写“自定义”。不要同时出现“我有自己的故事想法”“手动输入”“其他”等重复入口。

示例（如果用户选了"重生"）：
- "996打工人猝死后穿越成古代昏君，皇后逼宫要钱"
- "高考落榜生重生回高三第一天，这次要逆天改命"
- "被渣男害死的女总裁重生到婚礼当天，手撕全场"
- "外卖小哥穿越成将军府废物世子，开局就被下毒"
- 自定义

示例（如果用户选了"悬疑"）：
- "绿皮火车上，发现对面坐着的人是A级通缉犯"
- "接到婆婆电话催做饭，脚边躺着丈夫的尸体"
- "密室醒来，手机倒计时显示还剩12小时"
- "每天收到一张照片，拍的都是自己当天的行踪"
- 自定义

> 注意：生成的故事选项必须具体、有画面感、自带冲突。不要写抽象的描述，要写"一句话就能看到画面"的故事梗概。

---

### 第二步：生成分镜表格

根据用户选择的类型和故事，生成结构化的分镜表格供确认：

| 时间段 | 镜头类型 | 画面内容 | 对白/旁白 | 情绪/氛围 | 钩子元素 |
|--------|----------|----------|-----------|-----------|----------|
| 0-3秒  | 特写/快推 | [具体场景] | [开场对白] | [情绪] | [冲突点] |
| 3-6秒  | 中景/切换 | [身份展示] | [内心独白] | [反差] | [信息差] |
| 6-9秒  | 近景/运镜 | [矛盾激化] | [质问/宣言] | [张力] | [悬念] |
| 9-12秒 | 大特写/定格 | [反转收尾] | [点题/留白] | [冲击] | [记忆点] |

---

### 第三步：生成完整提示词

用户确认表格后，将其转化为详细的分镜脚本（即最终输出形态），格式见下方"提示词输出格式"。

---

### 第四步：用户确认与调整

展示生成的提示词，询问用户是否满意。如果用户提出修改意见，根据反馈调整后重新输出。

---

### 第五步：生成视频

**⚠️ 用户确认后，直接调用即梦 Agent 的「生成视频」功能，将提示词逐段生成为视频片段。**

确认无误后，自动将提示词转化为实际视频。无需用户额外操作。

## 核心写作规律

基于 13 部爆款短剧分析总结的黄金法则：

### 1. 12秒黄金结构

| 时间段 | 功能 | 核心任务 |
|--------|------|----------|
| 0-3秒 | 钩子 | 制造"停下滑动"的冲击——视觉震撼或情感爆发 |
| 3-6秒 | 设定 | 揭示身份/处境，制造信息差 |
| 6-9秒 | 冲突 | 情绪推进、矛盾激化、张力拉满 |
| 9-12秒 | 悬念 | 反转或定格，留下"然后呢？"的好奇 |

### 2. 开场钩子公式

**强钩子 = 视觉冲击 + 情感冲突 + 信息差**

- **视觉冲击**：特写推进、色调突变、快切剪辑、特效叠加
- **情感冲突**：震惊、愤怒、不甘、恐惧、心碎、渴望
- **信息差**：身份反差、时空错位、能力觉醒、秘密揭露、荒诞反转

### 3. 镜头语言工具箱

| 镜头类型 | 用途 | 情绪效果 |
|----------|------|----------|
| 大特写（眼/手/道具） | 情绪捕捉 | 紧张、震惊、细腻 |
| 特写（面部） | 表情传达 | 愤怒、恐惧、不甘 |
| 近景（上半身） | 人物状态 | 压迫、亲密 |
| 中景（全身） | 动作展示 | 对峙、行动 |
| 全景 | 环境渲染 | 孤独、宏大、压抑 |
| 俯拍 | 揭示全貌 | 震撼、无助 |
| 仰拍 | 权力/压迫 | 威严、恐惧 |

**运镜技巧**：
- 快速推进/急推：制造紧张感、突发感
- 慢动作：强化关键时刻的仪式感
- 360度环绕：展现被围攻的压迫感
- 快切剪辑（0.5-0.8秒/镜）：信息密度、节奏感
- 希区柯克变焦（Dolly Zoom）：眩晕、认知崩塌
- 主观视角（POV）：代入感、发现感
- 跟拍/侧跟平移：紧迫感、动态感

### 4. 对白策略

**原则：内心独白 > 直接对话**

- 前3秒必须有声音（对白/旁白/音效）
- 每句对白不超过10个字（3秒1句原则）
- 使用短句、反问、感叹制造冲突
- 留白悬念（话说一半、欲言又止）

**高效对白模式**：
- 质问式："说好的三百万两黄金呢？陛下难道要食言不成？！"
- 震惊式："我去，我刚才不是在开周会吗？"
- 宣言式："这一世，我要大富大贵！"
- 揭秘式："世子爷……猝死了！"
- 反差式："医生！我要给他绝育！"

### 5. 情绪细节词库

| 情绪 | 身体细节描写 |
|------|-------------|
| 紧张 | 手指颤抖、食指不规律敲击大腿、吞咽口水、额头冷汗 |
| 愤怒 | 咬牙、握拳、眼神凌厉、甩袖拍案、身体前倾 |
| 震惊 | 瞳孔放大/微颤、嘴唇微张、身体僵硬、下意识后退 |
| 不甘 | 眉头紧锁、指关节发白、眼眶泛红、凄惨冷笑 |
| 恐惧 | 睁不开眼、脸色苍白、牙齿打颤、眼神从疑惑转为深层恐惧 |
| 心碎 | 颤抖的手想触碰又收回、眼神充满不可置信 |
| 冷静（反差） | 面无表情、语调平稳得令人发指、步履均匀 |

### 6. 类型化钩子模板

**重生/穿越类**：
- 开场：现代场景突然切换到古代/异世界
- 钩子：身份错位（"我刚才不是在开周会吗？"）
- 冲突：前世记忆 vs 当前身份，现代思维 vs 古代规则
- 结尾：觉醒宣言或震惊定格

**豪门/复仇类**：
- 开场：身份对峙场景，华丽环境
- 钩子：真假身份揭露，地位反差
- 冲突：两人无声对峙，暗流涌动
- 结尾：联手悬念或反击预告

**修仙/玄幻类**：
- 开场：背叛围攻场景，法术光芒汇聚
- 钩子：师徒反目，恩将仇报
- 冲突：被围攻的压迫感，不甘与愤怒
- 结尾：黑屏前的悬念（重生？觉醒？）

**悬疑/惊悚类**：
- 开场：压抑环境渲染（荒漠、密闭空间、昏暗灯光）
- 钩子：危险人物出现/惊悚真相揭露
- 冲突：生死悬念、致命对视
- 结尾：戛然而止的心跳声/定格在恐怖画面

**末世/灾难类**：
- 开场：日常场景突然异变（天气骤变、异象降临）
- 钩子：从疑惑到恐惧的情绪转变
- 冲突：生存危机逼近，无处可逃
- 结尾：觉醒前兆或灾难全貌

**校园/励志类**：
- 开场：困境展示（贫穷、孤独、压力）
- 钩子：梦想宣言与现实差距的碰撞
- 冲突：分数不够、资源匮乏、时间紧迫
- 结尾：不甘与斗志的定格

**恋爱/情感类**：
- 开场：意外重逢的震撼瞬间
- 钩子：多年未见的心碎质问
- 冲突：物是人非、满身伤痕、无法触碰
- 结尾：未说出口的话，定格在沉默中

**恋爱奇幻类**：
- 开场：荒诞设定的视觉冲击
- 钩子：身份变异的爆笑/震惊
- 冲突：前任关系 + 奇幻设定的化学反应
- 结尾：出人意料的"报复"行动

**无限流类**：
- 开场：宏大的规则暗示（时间逆流、宇宙异象）
- 钩子：无限自由的诱惑展示
- 冲突：享乐表象下的代价暗示
- 结尾：规则揭示，循环开启

**古装悬疑类**：
- 开场：繁华场所的日常（教坊司、酒楼）
- 钩子：突发命案打破平静
- 冲突：层层传递的惊恐，快速升级
- 结尾：荒诞/恐怖的死亡现场定格

## 提示词输出格式

最终输出的提示词必须严格遵循以下格式：

```
[剧名]

类型：[类型]
角色：[角色设定，包括外貌、服装、气质描述]

**0-N秒：【[场景标题]】**
- **画面：** [镜头运动 + 环境 + 视觉细节，要具象化]
- **分镜：** [镜头类型 + 构图 + 运镜说明]
- **对白（[角色名/内心独白/旁白]）：** "[具体台词]"
- **音效：** [背景音/特效音]（如需要）
- **动作：** [角色肢体动作]（如需要）
- **视觉细节/特效：** [字幕、UI特效、色调变化]（如需要）

[重复以上结构，覆盖完整12秒]
```

**写作铁律**：

1. 场景描述必须具象化
   - ❌ "豪华的房间"
   - ✅ "金碧辉煌的欧式客厅，水晶吊灯投下冷白色光芒，大理石地面反射出人影"

2. 画面必须有动态感
   - ❌ "主角站在那里"
   - ✅ "镜头从下往上急速推向主角震惊的脸部，瞳孔微颤，背景虚化"

3. 对白必须制造冲突
   - ❌ "我很生气"
   - ✅ "说好的三百万两黄金呢？陛下难道要食言不成？！"

4. 分镜必须精准指导
   - ❌ "拍主角"
   - ✅ "中景 → 脸部大特写，采用希区柯克式变焦，背景拉伸营造眩晕感"

5. 每个时间段必须标注情绪功能
   - 冲突爆发 / 身份揭示 / 情绪转折 / 悬念收尾

## 质量检查清单

生成提示词后，必须逐项确认：

- [ ] 前3秒有强烈的视觉或情感冲击（能让人停下滑动）
- [ ] 每个时间段都有明确的情绪递进（不平淡）
- [ ] 对白简短有力，每句不超过10个字
- [ ] 镜头语言丰富，至少包含3种不同景别
- [ ] 至少有2个以上的悬念钩子
- [ ] 结尾留白，引发"然后呢？"的好奇
- [ ] 场景描述具象化，可直接用于视频生成
- [ ] 符合所选类型的核心冲突模式
- [ ] 有"卧槽"瞬间（反转/震惊/爆笑）

### 范例 1：《帝王觉醒录》（穿书）

**时长：** 约12秒
**场景：** 古代皇宫寝殿（金碧辉煌，烛光摇曳，气氛压抑）
**角色：**
- 皇帝（男主）：身穿黑色金绣龙袍，神情从懵懂转为震惊。现代人灵魂穿越。
- 皇后（魏薇）：妆容美艳犀利，头戴繁重金饰，气场咄咄逼人。
- 太监（王公公）：站在角落，一脸惊恐。

**0-4秒：【冲突爆发】**
- **画面：** 镜头从下往上急速推向皇后（魏薇）的脸部特写，她眼神凌厉，充满质问。随后快速剪辑至她全身画面，她愤怒地甩袖或拍案。
- **分镜：** 中景 -> 脸部大特写。
- **动作：** 皇后身体前倾，压迫感十足。
- **对白（皇后）：** （语速极快，咄咄逼人）"说好的三百万两黄金呢？陛下难道要食言不成？！"

**4-8秒：【现代思维错乱】**
- **画面：** 镜头猛地切到坐在龙椅上的皇帝（男主）。他一脸茫然，眼神游离，完全不在状态。背景虚化，强调他与周围环境的格格不入。
- **分镜：** 皇帝正面上半身镜头 -> 皇帝的主观视角（看着自己这身陌生的古装）。
- **动作：** 皇帝下意识地摸了摸身上的龙袍，眉头紧锁，眼神中透出清澈的愚蠢。
- **对白（皇帝）：** （一脸懵逼，现代口语语气）"什么大军？什么黄金？……我去，我刚才不是在开周会吗？"

**8-12秒：【觉醒与反转】**
- **画面：** 镜头急速变焦（Dolly Zoom，希区柯克式变焦），背景拉伸，营造眩晕感。皇帝猛地从龙椅上弹起。
- **分镜：** 皇帝震惊的面部特写 -> 快速闪回现代会议室画面（1秒叠化） -> 回到古代龙袍特写。
- **音效：** 轰鸣声或重音"Duang"的一声，配合心跳声。
- **对白（皇帝/内心独白）：** （极其震惊，看着双手）"这龙袍……我竟然重生了？！还成了个皇帝？！"
- **结尾画面：** 画面定格在皇帝不可思议的表情上。

### 范例 2：《时停千年：我被困在同一天》（无限流）

**0-2秒：悬念开场，时空凝滞**
- **画面：** 屏幕中央出现一个巨大的、散发着星云光芒的沙漏，沙子逆流而上。地球在沙漏背景中缓缓转动，镜头由远及近快速推进。
- **对白（旁白）：** "如果你的时间，每天都会重置……"
- **分镜：** 宏大远景切入，给人一种宇宙秩序被打乱的宿命感。

**2-5秒：豪车炸街，金钱自由**
- **画面：** 画面瞬间切到繁华都市夜景。一辆挂着"22222"炸弹号牌的红色法拉利跑车伴随巨大的引擎轰鸣声甩尾漂移，稳稳停在镜头前。男主角（一身白衬衫，神色从容且带着坏笑）推门下车。
- **对白（旁白）：** "你将不再为钱发愁，尽情享乐。"
- **分镜：** 侧面中景拍摄跑车入场，切男主下车的腿部特写（皮鞋、修身西裤），再切男主自信的近景面部特写。

**5-9秒：左拥右抱，名媛环绕**
- **画面：** 画面分割成上下两层。上层是男主在名媛堆里意气风发地走过；下层是无数身着晚礼服、妆容精致的名门闺秀对着男主尖叫、捂嘴欢呼。男主随手搂住两名美女，三人大笑着走向镜头。
- **对白（旁白）：** "和无数名门闺秀一夜疯狂，而不计后果。"
- **分镜：** 快节奏多画面平铺，展现出人群的喧闹和男主的主角光环。灯光闪烁，营造出迷乱的派对氛围。

**9-12秒：真相揭露，循环开启**
- **画面：** 画面色调突然冷峻。男主俯身贴近一个女孩的耳边，神情变得玩世不恭甚至有些冰冷。镜头快速掠过一双军靴踏下直升机的画面。
- **对白（旁白）：** "因为你知道，第二天，一切都会重置。"
- **分镜：** 男主侧脸特写切到黑色背景的标题，最后一帧定格在剧名《时停千年》。

**0-2秒：近景/特写**
- **画面：** 一名身穿笔挺深蓝色空乘制服的年轻男子（主角），原本正拉着行李箱在机场外行走。他突然感到一阵刺骨的寒意，身体猛地打了个冷颤，下意识地用双臂紧紧环抱住胸口。
- **对白（内心独白）：** 怎么突然这么冷啊？

**2-4秒：中景/仰拍**
- **画面：** 男子停下脚步，疑惑地抬起头望向灰暗的天空。几朵硕大的雪花落在他冷峻的脸上。镜头切换，天空中开始飘落密集的鹅毛大雪。
- **对白（惊诧）：** 下……下雪了？

**4-6秒：全景/空镜**
- **画面：** 镜头拉远，展现机场航站楼外的宏大场景。原本晴朗的天空瞬间被厚重的黑云笼罩，暴雪如瀑布般倾泻而下。机场巨大的电子显示屏在风雪中闪烁。

**6-8秒：特效镜头/远景**
- **画面：** 天空中，一架民航客机正试图穿越厚重的云层，但紧接着，一层诡异的冰晶迅速覆盖了整个机身，飞机在风雪中剧烈颠簸，引擎发出沉闷的轰鸣。

**8-10秒：近景/反应镜头**
- **画面：** 镜头切回主角。他被剧烈的寒风吹得几乎睁不开眼，睫毛和眉毛上迅速凝结出白霜。脸色苍白，眼神中从疑惑转为深层的恐惧。
- **对白：** 好冷……

**10-12秒：特写/拉开序幕**
- **画面：** 主角猛地伸出一只手，似乎想要捕捉空气中那股不寻常的寒流。特写他惊恐而逐渐变得坚毅的眼神。
- **画面处理：** 画面在强风雪中逐渐模糊，最后定格在男子那张被冰霜覆盖、充满觉醒前兆的脸庞。

## 常见问题

**Q：如何判断钩子够不够强？**
A：问自己三个问题：
1. 前3秒能让人停下滑动吗？
2. 看完12秒会想知道后续吗？
3. 有没有"卧槽"的瞬间？

**Q：对白太长怎么办？**
A：遵循"3秒1句"原则，超过10个字就要拆分或删减。

**Q：如何避免套路化？**
A：在经典模板上加入：
- 意外的身份设定（如前男友变狗）
- 反常识的情节转折（如杀人后接婆婆电话）
- 独特的视觉符号（如百叶窗牢笼阴影、逆流沙漏）
- 荒诞与恐怖的混搭（如穿着抹胸猝死的世子）

**Q：什么样的结尾最有效？**
A：四种高效结尾：
1. 悬念式：定格在关键动作前（被剑刺穿→黑屏）
2. 反转式：揭示意外信息（婚纱照沾血）
3. 宣言式：主角立下flag（"这一世，我要大富大贵！"）
4. 爆笑式：出人意料的行动（"医生！我要给他绝育！"）

**Q：色调如何选择？**
A：
- 悬疑/惊悚：冷色调（墨绿、深蓝）+ 局部暖光对比
- 重生/穿越：前世暗淡 vs 今生明亮的对比
- 恋爱/情感：偏暖滤镜回忆 vs 冷白现实
- 末世/灾难：从正常色调急转灰暗
- 豪门：高饱和度、强光、金色调

## 使用建议

1. **首次使用**：建议选择经典类型（重生/豪门/修仙），更容易掌握套路
2. **进阶创作**：尝试类型混搭（如"校园+悬疑"、"恋爱+末世"）
3. **爆款复刻**：可以参考上方范例，替换人设和场景
4. **持续优化**：根据生成效果，不断调整钩子强度和节奏
