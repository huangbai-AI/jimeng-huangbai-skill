# Jimeng Skills

这个仓库用于维护即梦 Agent 可上传使用的 Markdown 技能文档。

## 技能列表

| 文件 | 用途 |
| --- | --- |
| `Cover.md` | 生成小红书、抖音、视频号、B 站等平台的高点击封面图。 |
| `Product Ad.md` | 根据产品素材和需求，生成适合 Seedance 2.0 的商业产品广告 JSON 提示词。 |
| `Short Drama.md` | 生成短剧开头：先做角色形象设定图，再生成 12 秒强钩子分镜和视频提示词。 |

## 使用方式

1. 打开即梦 Agent。
2. 上传需要使用的 `.md` 技能文档。
3. 按技能文档内的流程操作，例如选择选项、上传素材、确认角色图或分镜。
4. 确认后再生成图片或视频。

## 约定

- 每个技能都是单独的 Markdown 文件。
- 文件名保持简短，便于上传和识别。
- 技能内流程优先使用即梦 Agent 的选项能力，让用户做关键选择。
- 生成视频时，如技能要求参考图，必须把参考图作为输入，不只写在提示词里。
